# Anchor Vault
A simple Solana Anchor program that manages deposits and withdrawals through a secure vault account.

⚡ Overview

This program demonstrates how to:

- Create a vault account on Solana
- Deposit SOL or tokens into the vault
- Withdraw from the vault securely
- Manage account ownership and permissions with Anchor

It’s designed as a minimal example to understand state management, PDAs, and cross-program invocations (CPIs) in Anchor.
